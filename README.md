# EAE Digital Portfolio - Jaron Chew Kai Xin

Welcome to the source code repository for my digital portfolio. This website was developed and deployed as part of my Early Admissions Exercise (EAE) application to Ngee Ann Polytechnic. 

## 🎯 Objective
This portfolio serves as a centralized showcase of my technical skills, certifications, and passion for technology. Instead of using a drag-and-drop website builder, I chose to code and host this portfolio manually to demonstrate my foundational understanding of web development, version control (Git), and deployment pipelines.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript
* **Version Control:** Git & GitHub
* **Hosting:** GitHub Pages
* **Design & Prototyping:** Draw.io (Flowcharts & User Flows)

## 📁 Repository Structure
* `/images/` & `/videos/` - Media assets for the UI and project demonstrations.
* `/Certifications/` - Scans of my technical achievements (Python, App Creation, Robotics, CoderZ, etc.).
* `index.html` - The main entry point of the application.
* `style.css` - Custom styling and responsive design.
* `script.js` & `data.js` - Dynamic functionality and data rendering.
* `DESIGN.md` - System architecture and design documentation.
